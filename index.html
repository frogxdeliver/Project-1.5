HTML Forms

HTML Forms are used to collect data from users who visit your website. HTML allows for various different input types such as email, password, checkbox, etc.
Below is an image that depicts an example of an HTML form.


An HTML Form allows you to make basic HTTP (HyperText Transfer Protocol) requests, such as GET and POST.

GET Request

A GET request is used to request data from a specified source.
The request goes to a URL and has no body.
Form parameters are attached at the end of the url as a query string.

POST Request

A POST request is used to send data to a server to create or update a resource.
The data is stored in the body of the HTTP request.
